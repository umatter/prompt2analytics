## Detailed Gap Analysis: HIGH Priority Methods

### ANOVA (`anova`, `aov`)

**Description:** Analysis of Variance for comparing group means
**Category:** ANOVA
**Priority Score:** 95/100
**Complexity:** Medium
**Documentation:** https://stat.ethz.ch/R-manual/R-devel/library/stats/html/anova.html

**Existing Related Components:**
- `run_ols` in `regression/ols.rs` - provides RSS, residuals, F-statistic
- `f_test_p_value` in `traits/estimator.rs` - F-distribution p-values
- Design matrix construction in `linalg/design.rs`

**Implementation Notes:**
- One-way ANOVA: compare means across groups using F-test
- Two-way ANOVA: main effects + interaction
- Need: SS decomposition (between, within, total), df calculation
- Output: ANOVA table (SS, df, MS, F, p-value)

**Command:** `/implement_metrics https://stat.ethz.ch/R-manual/R-devel/library/stats/html/aov.html`

**Description:** Fit a linear model by generalized least squares
**Category:** Regression

### METHOD: cor.test (Correlation Test)

**Description:** Test for association/correlation between paired samples
**Category:** Hypothesis Testing
**Priority Score:** 92/100
**Complexity:** Simple
**Documentation:** https://stat.ethz.ch/R-manual/R-devel/library/stats/html/cor.test.html

**Existing Related Components:**
- `correlation_matrix` in `stats/correlation.rs` - calculates correlation
- `t_test_p_value` in `traits/estimator.rs` - p-value calculation
- Missing: Fisher's z-transformation for CI, Spearman/Kendall tests

**Implementation Notes:**
- Pearson: t = r * sqrt((n-2)/(1-r²)), df = n-2
- Spearman: rank correlation, then Pearson formula
- Kendall: O(n²) or O(n log n) algorithm
- CI via Fisher z-transformation: z = 0.5 * ln((1+r)/(1-r))

**Command to implement:**
```
/implement_metrics https://stat.ethz.ch/R-manual/R-devel/library/stats/html/cor.test.html
```

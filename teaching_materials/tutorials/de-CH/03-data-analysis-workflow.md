# Vollständiger Datenanalyse-Workflow

## Lernziele

Nach Abschluss dieses Tutorials können Sie:
- [ ] Ein Datenanalyse-Projekt planen
- [ ] Eine vollständige Analyse vom Laden bis zum Bericht durchführen
- [ ] Mehrere Analysetechniken kombinieren
- [ ] Ergebnisse effektiv präsentieren

## Voraussetzungen

- Abgeschlossen: [CLI-Tutorial](01-cli-getting-started.md) oder [Chat-Interface-Anleitung](02-chat-interface-guide.md)
- Verständnis grundlegender statistischer Konzepte
- Beispieldatensätze aus `teaching_data/`

## Überblick

Dieses Tutorial führt durch ein vollständiges Datenanalyse-Projekt mit den Beispieldatensätzen. Wir behandeln:
1. Problemdefinition
2. Daten laden und bereinigen
3. Explorative Datenanalyse
4. Statistische Modellierung
5. Ergebnisinterpretation
6. Berichterstattung

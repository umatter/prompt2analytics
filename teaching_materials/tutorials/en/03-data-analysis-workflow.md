# Complete Data Analysis Workflow

## Learning Objectives

After completing this tutorial, you will be able to:
- [ ] Plan a data analysis project
- [ ] Execute a full analysis from data loading to reporting
- [ ] Combine multiple analytical techniques
- [ ] Present findings effectively

## Prerequisites

- Completed either the [CLI Tutorial](01-cli-getting-started.md) or [Chat Interface Guide](02-chat-interface-guide.md)
- Understanding of basic statistical concepts
- Sample datasets from `teaching_data/`

## Overview

This tutorial walks through a complete data analysis project using the sample datasets. We'll cover:
1. Problem definition
2. Data loading and cleaning
3. Exploratory data analysis
4. Statistical modeling
5. Results interpretation
6. Reporting findings

# Using the p2a Chat Interface

## Learning Objectives

After completing this tutorial, you will be able to:
- [ ] Launch and navigate the p2a web interface
- [ ] Load and explore datasets using natural language
- [ ] Request statistical analyses through conversation
- [ ] Interpret regression results and visualizations
- [ ] Build a complete analysis workflow through chat

## Prerequisites

- p2a web application running (see Setup section)
- Web browser (Chrome, Firefox, or Safari recommended)
- Sample datasets from `teaching_data/`

## Why the Chat Interface?

The chat interface allows you to perform sophisticated data analysis without writing code. Simply describe what you want to do in plain English, and the system will:
- Load and process your data
- Run appropriate statistical methods
- Generate visualizations
- Explain the results

This is ideal for:
- Business analysts without programming background
- Quick exploratory data analysis
- Learning statistical concepts hands-on

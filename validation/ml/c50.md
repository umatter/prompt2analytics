# C5.0 Decision Trees

## Method

C5.0 is a decision tree / rule-based classifier that uses information gain ratio for splitting, pessimistic pruning, and optional boosting (AdaBoost-like trials). It is the successor to C4.5 by J. Ross Quinlan.

## R Package Reference

- **Package**: `C50` (v0.1.8+)
- **Function**: `C5.0(x, y, trials = 1, rules = FALSE, control = C5.0Control())`
- **CRAN**: https://cran.r-project.org/package=C50

## R Code for Expected Values

```r
library(C50)
set.seed(42)

n <- 200
x1 <- runif(n)
x2 <- runif(n)
x3 <- rnorm(n, 0, 0.5)
y <- factor(ifelse(x1 + x2 > 1.0, "1", "0"))
df <- data.frame(x1, x2, x3)

# Single tree
model <- C5.0(df, y, trials = 1)
summary(model)
pred <- predict(model, df)
cat("Training accuracy:", mean(pred == y), "\n")

# Boosted (10 trials)
model_boost <- C5.0(df, y, trials = 10)
pred_boost <- predict(model_boost, df)
cat("Boosted accuracy:", mean(pred_boost == y), "\n")

# Variable importance
C5imp(model_boost, metric = "usage")
```

## Rust Test Reference

- **File**: `crates/p2a-core/tests/validation_ml_advanced.rs`
- **Tests**:
  - `test_validate_c50_classification` - Single tree accuracy > 0.80
  - `test_validate_c50_boosted` - 10-trial boosted accuracy > 0.80
  - `test_validate_c50_predict` - Out-of-sample accuracy > 0.70

## Tolerance

- Training accuracy: > 0.80 (separable synthetic data)
- Out-of-sample accuracy: > 0.70
- Variable importance: signal features (x1, x2) > noise (x3)

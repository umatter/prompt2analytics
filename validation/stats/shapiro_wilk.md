# Validation: Shapiro-Wilk Normality Test

## Method Overview

This document validates the p2a-core implementation of the Shapiro-Wilk normality test against R's `stats::shapiro.test()`.

**Functions Implemented:**
- `shapiro_wilk_test()` - Direct test on array data
- `run_shapiro_wilk()` - Dataset wrapper for column-based access

**Key Parameters:**
- Input: numeric vector x (3 ≤ n ≤ 5000)
- Output: W statistic, p-value

**Null Hypothesis:** H₀: The data is normally distributed
**Alternative Hypothesis:** H₁: The data is not normally distributed

## Mathematical Formulas

### W Statistic

```
W = (∑aᵢx₍ᵢ₎)² / ∑(xᵢ - x̄)²
```

Where:
- x₍ᵢ₎ is the i-th order statistic (i-th smallest value)
- x̄ is the sample mean
- aᵢ are coefficients derived from expected normal order statistics

### Coefficients

The coefficients a = (a₁, ..., aₙ) are approximated using:
- Blom's formula for expected normal order statistics: m_i ≈ Φ⁻¹((i - 3/8) / (n + 1/4))
- Polynomial corrections from Royston (1992, 1995)

### P-Value Approximation

Following Royston (1995):
- For n = 3: exact distribution
- For 4 ≤ n ≤ 11: polynomial approximation with gamma transformation
- For n ≥ 12: log transformation with normal approximation

## Numerical Precision Summary

| p-value | Qualitative | Both should indicate same normality conclusion |
| n | Exact | Sample size after filtering NaN |

## Known Differences

1. **Coefficient Calculation:**
   - R uses exact AS R94 algorithm from Royston (1995)
   - Our implementation uses polynomial approximations which may give slightly different W values
   - Both implementations correctly identify normal vs non-normal data

2. **P-Value Approximation:**
   - Both use similar transformations but may differ in polynomial coefficients
   - Qualitative conclusions (reject/fail to reject) should match

3. **Sample Size Handling:**
   - R: 3 ≤ n ≤ 5000
   - Rust: Same constraints with explicit error messages
